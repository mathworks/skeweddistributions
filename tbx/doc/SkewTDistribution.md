# SkewT Distribution

The **Skew‑t distribution** generalizes both the **Student's t distribution**
and the **skew‑normal distribution**, allowing for **asymmetry** and
**heavy tails** simultaneously.

This class implements the skew-t distribution as a subclass of `prob.ToolboxFittableParametricDistribution`, making it compatible with MATLAB's
Statistics and Machine Learning Toolbox.

## Class

```matlab
prob.SkewTDistribution
```

## Parameters
| Name | Description | Constraint |
|------|-------------|------------|
| $\xi$ | Location | $\xi \in \mathbb R$ |
| $\omega$ | Scale | $\omega > 0$ |
| $\alpha$ | Shape (skewness) | $\alpha \in \mathbb R$ |
| $\nu$ | Degrees of freedom | $\nu > 0 $ |

## Support
$$x \in \left(-\infty, \infty\right)$$

## Probability Density Function (PDF)

Let

$$
z = \frac{x - \xi}{\omega}
$$

For finite $\nu$, the PDF is:

$$
f(x) =
\frac{2}{\omega}
t_\nu(z)
T_{\nu+1}\!\left(
\alpha z \sqrt{\frac{\nu+1}{z^2+\nu}}
\right)
$$

where:

- $t_\nu$ is the Student's t PDF
- $T_\nu$ is the Student's t CDF

For $\nu = \infty$, the distribution reduces to the skew‑normal PDF.

## Cumulative Distribution Function (CDF)

- For $\nu = \infty$, the skew‑normal CDF is used
- For integer $\nu$, the CDF is computed via the bivariate t distribution
- Otherwise, numerical integration is used for robustness

## Moments

### Mean

The mean exists only for $\nu > 1$. In that case:

$$
\mathbb E[X] = \xi + \omega \delta\sqrt{\frac{\nu}{\pi}}\frac{\Gamma\left(\frac{\nu-1}{2}\right)}     {\Gamma\left(\frac{\nu}{2}\right)}
$$

where

$$\delta = \frac{\alpha}{\sqrt{1+\alpha^2}}$$

For $\nu = \infty$, this reduces to the skew‑normal mean.

## Variance

The variance exists only for $\nu > 2$. In that case:

$$
\mathrm{Var}(X)=\omega^2\left(\frac{\nu}{\nu-2}- \mu^2\right)
$$

where $\mu$ is the standardized mean contribution.

## Construction

Fixed-parameter construction

```matlab
pd = makedist("SkewT", xi, omega, alpha, nu); 
```
If constructed explicitly:

- Parameters are treated as fixed
- ParameterIsFixed = [true, true, true, true]
- ParameterCovariance = 0

## Fitting to Data

The skew‑t distribution supports maximum likelihood estimation:

```matlab 
pdHat = fitdist(x, "SkewT"); 
```

## Example Usage

```matlab
% Create a distribution object
pd = makedist("SkewT", 0, 1, 3, 5);

% Evaluate the PDF
x = linspace(-5, 5, 200);
y = pdf(pd, x);

% Random sampling
r = random(pd, 10000, 1);

% Fit a distribution to data
pdHat = fitdist(r, 'SkewT');

% plot the fitted results
figure(Color = "w");
histogram(r, 100, Normalization = "pdf")
hold on; 
x = linspace(-1, 5, 200);
y = pdf(pdHat, x);
line(x, y, LineWidth = 2)
title('SkewT Distribution')
```

## Random Number Generation

Random variates are generated using a scale‑mixture representation:
```matlab
r = random(pd, n, 1); 
```
- Skew‑normal latent variable
- Scaled by a $\chi^2/\nu$ random variable for finite $\nu$
## Notes

- Combines skewness and heavy tails in a single model
- Reduces cleanly to skew‑normal and Student's t as special cases
- Suitable for financial, environmental, and robust modeling
