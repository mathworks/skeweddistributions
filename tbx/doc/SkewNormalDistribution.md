# Skew Normal Distribution

The **Skew-Normal distribution** is a generalization of the normal distribution
that introduces **asymmetry** through a shape parameter. It retains many
tractable properties of the normal distribution while allowing for skewness.

This class implements the skew-normal distribution as a subclass of `prob.ToolboxFittableParametricDistribution`, making it compatible with
MATLAB's Statistics and Machine Learning Toolbox.

---

## Class
```matlab
prob.SkewNormalDistribution
```

## Parameters
| Name | Description | Constraint |
|------|-------------|------------|
| $\xi$ | Location | $\xi \in \mathbb R$ |
| $\omega$ | Scale | $\omega > 0$ |
| $\alpha$ | Shape (skewness) | $\alpha \in \mathbb R$ |

- $\alpha = 0$ reduces the distribution to a standard normal (up to location/scale).
- Positive alpha induces right skewness.
- Negative alpha induces left skewness.

## Support

$x \in (-\infty, \infty)$

## Probability Density Function (PDF)
Let

$$ u = \frac{x - \xi}{\omega} $$

The skew-normal PDF is defined as:

$$
f(x)=\frac{2}{\omega}\phi(z)\Phi(\alpha z)
$$

where:

- $\phi(\cdot)$ is the standard normal PDF
- $\Phi(\cdot)$ is the standard normal CDF

## Cumulative Distribution Function (CDF)

The CDF is given by:

$$
F(x)=\Phi(z)-2T(z, \alpha)
$$

where $T(h, a)$ is Owen's T function, used to account for skewness. This implementation provides a custom numerical evaluation of Owen's T function.

## Moments

### Mean
$$
\mathbb{E}[X] = \xi + \omega\delta\sqrt{\frac{2}{\pi}}
$$

### Variance
$$
Var(X) = \omega^2\left(1-\frac{2\delta^2}{\pi}\right)
$$

## Construction

Fixed-parameter construction

```matlab
pd = makedist("SkewNormal", xi, omega, alpha); 
```
If constructed explicitly:

- Parameters are treated as fixed
- ParameterIsFixed = [true, true, true]
- ParameterCovariance = 0

## Fitting to Data

The distribution supports maximum likelihood estimation via fitdist.

```matlab 
pdHat = fitdist(x, "SkewNormal"); 
```

## Example Usage

```matlab
% Create a distribution object
pd = makedist("SkewNormal", 0, 1, 5);

% Evaluate the PDF
x = linspace(-5, 5, 200);
y = pdf(pd, x);

% Random sampling
r = random(pd, 10000, 1);

% Fit a distribution to data
pdHat = fitdist(r, 'SkewNormal');

% plot the fitted results
figure(Color = "w");
histogram(r, 100, Normalization = "pdf")
hold on; 
x = linspace(-1, 5, 200);
y = pdf(pdHat, x);
line(x, y, LineWidth = 2)
title('SkewNormal Distribution')
```

## Random Number Generation
Random values are generated using a latent-variable representation based on correlated normal variables:

```matlab
r = random(pd, n, 1); 
```

This approach avoids numerical inversion and is efficient for large samples.

## Notes

- The skew-normal distribution is continuous and unimodal
- Skewness is controlled smoothly by $\alpha$
- Owen's T function is evaluated numerically for stability
- When $\alpha = 0$, the distribution reduces exactly to a normal distribution
